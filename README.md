## Usage

1. Download repo:
```sh
git clone https://github.com/AaronE90/Inbox-Cleaner.git
```
2. Change directory:
```sh
cd Inbox-Cleaner
```
3. Install Requirements:
```sh
python3 -m pip install -r requirements.txt
```
4. Run script:
```sh
python3 IMap-script.py
```
